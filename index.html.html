Estaremos hablando de las computadoras, son dispositivos electrónicos diseñados para realizar una amplia variedad de tareas informáticas de manera portátil. Están compuestas por una combinación de hardware y software que les permite ejecutar programas, procesar datos, conectarse a redes, y realizar muchas otras funciones. Las laptops son más compactas y livianas que las computadoras de escritorio, lo que las hace convenientes para su uso en movimiento. Generalmente incluyen una pantalla, un teclado, un touchpad o un dispositivo señalador similar (como un trackpad), y una batería recargable que permite su uso sin necesidad de estar conectadas a una fuente de alimentación eléctrica durante un tiempo limitado. Las laptops pueden variar en especificaciones y capacidades, desde modelos básicos adecuados para tareas simples como navegar por internet y procesar texto, hasta modelos más potentes destinados a tareas exigentes como edición de video, diseño gráfico o desarrollo de software. Su versatilidad las hace populares tanto para uso personal como profesional. Equipadas con una variedad de especificaciones de hardware, como procesadores, memoria RAM y almacenamiento, las laptops ofrecen un rendimiento versátil para una amplia gama de tareas informáticas, desde navegación web básica hasta procesamiento de datos intensivo. Además, su conectividad inalámbrica y los múltiples puertos de entrada y salida permiten la conexión a redes y dispositivos externos, proporcionando una experiencia informática completa y adaptable a las necesidades individuales del usuario.
